#  Smart Debt Minimizer

##  Description
This project is a web-based application that minimizes the number of transactions required to settle group expenses.

##  Features
- Dynamic user input
- API integration (fetching sample users)
- Greedy algorithm for optimization
- Clean and interactive UI

##  Technologies Used
- HTML
- CSS
- JavaScript
- REST API

##  Algorithm
The system calculates total expenses and equal share, then uses a greedy approach to match the highest debtor with the highest creditor to minimize transactions.

##  How to Run
1. Download the project
2. Open `index.html` in browser
3. Click buttons to test functionality
